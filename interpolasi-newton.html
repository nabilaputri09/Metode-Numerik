Ringkasan Interpolasi Newton

Interpolasi Newton membangun polinom interpolasi dari data titik (x‚ÇÄ, f(x‚ÇÄ)), (x‚ÇÅ, f(x‚ÇÅ)), ..., (x‚Çô, f(x‚Çô)) menggunakan beda terbagi (divided differences).

  • Beda terbagi pertama: f[x‚ÇÅ, x‚ÇÄ] = (f(x‚ÇÅ) ‚àí f(x‚ÇÄ)) / (x‚ÇÅ ‚àí x‚ÇÄ).
  • Beda terbagi kedua: f[x‚ÇÇ, x‚ÇÅ, x‚ÇÄ] = (f[x‚ÇÇ, x‚ÇÅ] ‚àí f[x‚ÇÅ, x‚ÇÄ]) / (x‚ÇÇ ‚àí x‚ÇÄ).
  • Bentuk polinom Newton derajat 1:
    P‚ÇÅ(x) = f[x‚ÇÄ] + f[x‚ÇÅ, x‚ÇÄ](x ‚àí x‚ÇÄ).
LATIHAN SOAL INTERPOLASI NEWTON
Konsep dasar dan perhitungan beda terbagi.
3 soal ‚Ä¢ latihan
Soal 1 Konsep dasar
Interpolasi Newton digunakan untuk...
Interpolasi Newton adalah metode untuk menyusun polinom yang melewati beberapa titik data. Dari polinom tersebut kita bisa mendekati nilai fungsi di titik lain. Jadi, jawabannya adalah mencari nilai pendekatan suatu fungsi berdasarkan data titik.
Soal 2 Beda terbagi derajat 1
Diberikan data titik berikut:

‚Ä¢ x‚ÇÄ = 1, dengan f(x‚ÇÄ) = 3
‚Ä¢ x‚ÇÅ = 2, dengan f(x‚ÇÅ) = 5

Gunakan interpolasi beda terbagi Newton derajat 1 untuk memperkirakan nilai f(1,5).
1. Hitung beda terbagi pertama:
f[x‚ÇÅ, x‚ÇÄ] = (f(x‚ÇÅ) ‚àí f(x‚ÇÄ)) / (x‚ÇÅ ‚àí x‚ÇÄ) = (5 ‚àí 3)/(2 ‚àí 1) = 2.

2. Polinom Newton derajat 1:
P‚ÇÅ(x) = f[x‚ÇÄ] + f[x‚ÇÅ, x‚ÇÄ](x ‚àí x‚ÇÄ) = 3 + 2(x ‚àí 1).

3. Substitusi x = 1,5:
P‚ÇÅ(1,5) = 3 + 2(1,5 ‚àí 1) = 3 + 2(0,5) = 3 + 1 = 4.

Jadi, nilai pendekatan f(1,5) adalah 4,0.
Soal 3 Beda terbagi kedua
Diberikan data titik berikut:

‚Ä¢ x‚ÇÄ = 0, dengan f(x‚ÇÄ) = 1
‚Ä¢ x‚ÇÅ = 1, dengan f(x‚ÇÅ) = 3
‚Ä¢ x‚ÇÇ = 2, dengan f(x‚ÇÇ) = 7

Hitung beda terbagi kedua f[x‚ÇÇ, x‚ÇÅ, x‚ÇÄ].
1. Beda terbagi pertama:
f[x‚ÇÅ, x‚ÇÄ] = (3 ‚àí 1)/(1 ‚àí 0) = 2
f[x‚ÇÇ, x‚ÇÅ] = (7 ‚àí 3)/(2 ‚àí 1) = 4

2. Beda terbagi kedua:
f[x‚ÇÇ, x‚ÇÅ, x‚ÇÄ] = (f[x‚ÇÇ, x‚ÇÅ] ‚àí f[x‚ÇÅ, x‚ÇÄ]) / (x‚ÇÇ ‚àí x‚ÇÄ) = (4 ‚àí 2)/(2 ‚àí 0) = 2/2 = 1.

Jadi, nilai beda terbagi kedua adalah 1.
